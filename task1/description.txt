1. ИНФОРМАЦИЯ О ИСТОЧНИКЕ ДАННЫХ И ПОРЯДКЕ ИХ ПОЛУЧЕНИЯ
- Информация о рейсах взята с сервиса flightStats (https://developer.flightstats.com);
- Данные, выданные api храняться в папке data/api/[arrivals|departures];
- Данные обработаны скриптом data/prepareData.js для удобного вывода в интерфейсе;
- Создан файл data/prepared/flights.js, содержащий обработанные данные.
2. ИНФОРМАЦИЯ О ИНТЕРФЕЙСЕ И ЗАДЕЙСТВОВАННЫХ ТЕХНОЛОГИЯХ
- Вывод данных осуществляется через шаблонизатор Handlebars;
